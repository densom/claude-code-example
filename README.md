# TI-84 Graphing Calculator

A Python implementation of a TI-84 graphing calculator featuring a familiar GUI interface and mathematical functionality.

## Features

- **Calculator Interface**: Authentic TI-84 button layout and display
- **Mathematical Functions**: Basic arithmetic, trigonometric functions, logarithms, and exponents
- **Graphing Capability**: Built-in graphing window for visualizing mathematical functions
- **Memory Display**: Running history of calculations and results

## Requirements

- Python 3.13+
- matplotlib
- numpy
- tkinter (included with Python)

## Installation

1. Clone the repository
2. Install dependencies using uv:
   ```bash
   uv sync
   ```

## Usage

Run the calculator:
```bash
python ti84_calculator.py
```

Or run the simple main script:
```bash
python main.py
```

## Controls

- **Number buttons (0-9)**: Input digits
- **Operators (+, -, ×, ÷, ^)**: Mathematical operations
- **Functions (SIN, COS, TAN, LOG, LN)**: Trigonometric and logarithmic functions
- **X²**: Square function
- **ENTER**: Calculate result
- **CLEAR**: Clear current input
- **DEL**: Delete last character
- **GRAPH**: Open graphing window
- **QUIT**: Exit calculator
